A acessibilidade na web

O que é a acessibilidade na web?

 A acessibilidade na web é uma forma de quebrar barreiras e permitir que todos, independente de terem ou não alguma deficiência, tenham acesso a web de forma compreensivel e com autonomia.

 Para criar algo acessivel é preciso, antes de tudo, saber que diferentes pessoas utilizam diferentes tipos de recursos e tecnologias assistivas, um exemplo disso é o fato de que a acessibilidade para uma pessoa cega e para uma pessoa surda serão diferentes, enquanto alguém com deficência visual precisa de um leitor de tela, uma pessoa com deficiência auditiva precisa de um tradutor para libras.



Por que ela é importante?

 Mais do que ter um design bonito, é importante que um site seja acessivel para todos os possiveis usuários do mesmo, pois, quando um site não é acessivel para todos, ele está impedidindo que milhares de pessoas tenham acesso a ele.

 Hoje em dia existem diversos recursos que ajudam a tornar um site acessivel e ter esse diferencial não só ajuda o seu site a ter vantagem contra concorrentes que não dão essa acessibilidade, como está cumprindo com a Lei Brasileira de inclusão (LBI) que exige que todos os sites, públicos e privados, estejam acessíveis.



Dicas de acessibilidade

 Como dito anteriormente, existem diversos recursos para tornar um site acessivel, por isso, alguns exemplos serão dados a seguir:

 1- Evitar usar cores para destacar algo em um texto. Muitas vezes são usadas palavras com cores diferentes para destaca-las, mas isso é um empecilho para pessoas daltônicas, por esse motivo é recomendado usar outros meios para eses destaques.

 2- Usar a propiedade (alt="") para dar uma descrição para fotos tambem é util, pois os leitores de tela podem descreve-las para quem tem deficiência visual.

 3- Cuido com animações infinitas, pois, por mais sutis que sejam, podem ser consideradas distrações para pessoas com TDAH (Transtorno de Déficit de Atenção com Hiperatividade).
